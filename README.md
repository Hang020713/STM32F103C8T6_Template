# About This Project

[![Banner Photo][banner-photo]](https://www.st.com/)
A custom STM32F103C8T6 PCB template designed for hand-soldering, prototyping, and embedded system development.
This board serves as a solid starting point for STM32-based projects, robotics systems, and embedded applications.

# 🚀 Getting Started

Requirements
* KiCad 8 (recommended)

Open the `.kicad_pro` file using KiCad 8 to access the full project including schematic and PCB layout.

---

# 🔧 Board Feature

STM32 Board have below features
- [x] Hand-solder friendly footprint design
- [x] 5V Voltage Regulator
- [x] Micro USB Connector
- [x] 1 x UART Interface
- [x] 1 x I2C Interface
- [x] BOOT Button
- [x] SWD Debugging Interface
- [x] 16 MHz External Crystal Oscillator
- [ ] USB-C Connector

---

# 🧠 Usage

STM32 have a deep potential. You could modify this board to extend more pins you want. Example:

* Expose additional GPIO Pins
* Add SPI peripherals
* Integreate sensors (IMU, barometer, etc.)
* Add motor drivers
